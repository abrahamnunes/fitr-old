The fitr package is a tool for fitting reinforcement learning (RL) models to behavioural data. Currently, it's in early stages of development, and only the Matlab package is presently functional.

Much credit is owed to Samuel Gershman for his mfit package from which I learned a great deal about implementing RL model fitting. Currently, the fitr Bayesian Model Selection code uses many of the functions from mfit, but an independent implementation is underway.

I am working on the python and Julia packages, but improving the quality of the Matlab version is currently my main priority.

Future goals include:

  • Documentation
  • Built-in gradient computations for common models
  • Bridging with tools such as SPM and VBA-Toolbox
  • Expanding base of built-in tasks
  • Improving speed by fitting subjects in parallel
  • Exploring GPU-acceleration
